# Project A

This repository contains a Vite + React + TypeScript starter. It includes:

- React 18 with Strict Mode enabled
- Vite for fast development builds and HMR
- TypeScript configuration tuned for modern browsers

## Available scripts

- `npm install` — install dependencies.
- `npm run dev` — start the Vite development server.
- `npm run build` — type-check the project and create a production build under `dist`.
- `npm run preview` — preview the production build locally.
